# AI Study App

This is an application for creating study guides using OpenAI's GPT-3 and text-to-speech (TTS) capabilities. It allows users to input study notes and generate a detailed study guide from them.

## Features

- Input study notes and generate a study guide.
- Play the generated study guide using text-to-speech (TTS).
- Control playback with options like play, pause, fast forward, and rewind.

### Obtaining an API Key for ChatGPT

To use ChatGPT, you'll need an API key from OpenAI. Here are the steps to obtain an API key:

1. Visit the OpenAI website at [https://beta.openai.com/](https://beta.openai.com/).

2. Sign in to your OpenAI account or create a new account if you don't have one.

3. Once you're logged in, navigate to the API section.

4. Create a new API key by following the instructions provided on the OpenAI platform.

5. Once you have your API key, set it as an environment variable named "OPENAI_API_KEY" on your local machine. This is essential for the AI Study App to access the OpenAI API.
